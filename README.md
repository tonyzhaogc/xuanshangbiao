# 商标子类选择系统

一个现代化、美观的商标子类选择和查询Web应用，支持45个商标分类的快速检索和导出。

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

### 🎯 三栏式布局设计
- **顶部**：45个商标大类横向展示，直观选择
- **左侧**：选中大类的详细内容（类似群、商品）
- **右侧**：已选中的商标明细列表

### 🎨 现代化UI设计
- 清新的配色方案（灰色主题）
- 流畅的动画效果和交互反馈
- 响应式设计，适配不同屏幕尺寸
- 优雅的卡片式布局和阴影效果

### 🔧 核心功能
- **快速选择**：点击大类即可查看详细内容
- **固定布局**：类似群列表固定，商品列表固定显示
- **实时搜索**：支持按编码或名称搜索商品
- **多选功能**：可同时选择多个商标
- **折叠展开**：类似群独立显示，无需滚动
- **分组显示**：右侧已选商标按大类自动分组

### 📤 导出功能
- **一键复制**：复制选中商标到剪贴板
- **导出CSV**：导出为CSV格式，兼容Excel
- **分类导出**：包含大类、类似群、商品编码、名称、备注

### 📊 数据特点
- **完整数据**：包含45个大类、494个类似群、10,158个商品
- **数据合并**：自动合并重复商品编码，显示所有别名
- **备注信息**：包含483条备注信息
- **简短名称**：使用4字简短名称，便于快速浏览

## 🚀 快速开始

### 方法1：本地服务器（推荐）

```bash

git clone https://github.com/tonyzhaogc/xuanshangbiao.git
cd xuanshangbiao

# 启动本地服务器

# 在浏览器中打开
# http://localhost:8000
```

### 方法2：直接打开

直接用浏览器打开 `index.html` 文件即可使用。

## 📖 使用指南

### 1. 选择商标大类
- 顶部显示45个商标大类
- 只显示大类编号和简短标题（如"07 机械设备"）
- 点击任意大类进入详细页面

### 2. 浏览类似群
- 左侧显示选中大类的所有类似群（4位编码）
- 类似群列表固定显示，独立滚动
- 点击类似群即可查看商品

### 3. 选择商品
- 右侧显示当前类似群的所有商品（6位编码）
- 商品列表固定显示，无需滚动寻找
- 点击商品即可选择或取消选择
- 支持搜索功能，快速定位商品

### 4. 管理已选商标
- 最右侧显示所有已选商标
- 按大类自动分组，便于查看
- 单个商标可点击删除按钮移除
- 支持一键清空所有选择

### 5. 导出数据
- 点击"导出"按钮下载CSV文件
- 点击"复制"按钮复制到剪贴板
- 导出包含完整信息（大类、类似群、商品编码、名称、备注）

## 💡 技术特点

- **纯前端实现**：无需后端服务器
- **现代CSS**：使用CSS变量、Flexbox、Grid布局
- **优雅降级**：支持旧版浏览器
- **性能优化**：懒加载、虚拟滚动（可选）
- **交互优化**：平滑动画、即时反馈
- **数据处理**：自动合并重复商品，显示所有别名

## 🔄 数据更新

如需更新数据，运行：

```bash
python3 scrape_data.py
```

数据抓取脚本会：
1. 从 https://www.zcw.com.cn/assets/tm-classes.js 获取最新数据
2. 使用简短名称映射表（4字名称）
3. 自动合并重复商品编码
4. 生成 trademark_data.json 文件
