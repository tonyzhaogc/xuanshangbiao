### 统计数据

### 合并策略
- 相同商品编码的商品合并为一个
- 多个名称用" | "连接
- 多个备注用" | "连接
- 保留去重后的所有名称和备注

## 导出功能
导出CSV文件时，合并后的商品会显示完整的名称（包含所有别名），例如：
```
大类,类似群,商品编码,商品名称,备注
28,2801,280044,力量训练器械 | 锻炼身体肌肉器械 | 锻炼身体器械,
```

## 优势
1. ✅ 消除数据冗余
2. ✅ 便于查看同一编码的不同别名
3. ✅ 减少重复选择
4. ✅ 导出时更清晰
