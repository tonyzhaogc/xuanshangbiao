# 商标子类选择系统

一个方便选择商标编码的Web应用，支持45个商标分类的快速检索和导出。

## 功能特性

- **完整的商标分类数据**：包含45个大类、494个类似群、10,587个商品
- **树状导航**：左侧树状结构快速浏览商标分类
- **实时搜索**：支持按编码或名称搜索
- **多选功能**：可同时选择多个商标分类
- **详情展示**：右侧面板显示选中商标的详细信息
- **复制功能**：一键复制选中商标信息
- **导出功能**：支持导出为CSV/XLS文件

## 数据来源

数据来源于 https://www.zcw.com.cn/trademark/classes2

## 安装和运行

### 方法1：直接打开HTML文件

直接用浏览器打开 `index.html` 文件即可使用。

### 方法2：使用本地服务器（推荐）

```bash
# Python 3
python3 -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000
```

## 使用说明

1. **浏览商标分类**
   - 左侧树状结构显示所有45个商标大类
   - 点击 ▶ 展开子分类，点击 ▼ 折叠

2. **搜索商标**
   - 在左侧搜索框输入关键词
   - 支持按编码（如 "07"、"0701"）或名称搜索

3. **选择商标**
   - 点击树状结构中的任意商标进行选择
   - 右侧面板显示选中商标的详细信息

4. **管理已选商标**
   - 右侧上方显示所有已选商标
   - 可以单独移除某个商标，或一键清空

5. **导出数据**
   - 点击"导出XLS"按钮下载CSV文件
   - 点击"复制选中"按钮复制到剪贴板

## 数据结构

商标分类分为三级：

- **大类**（2位编码）：如 01、07、25
- **类似群**（4位编码）：如 0701、0702
- **商品**（6位编码）：如 070008、070009

## 文件说明

- `index.html` - 主页面文件
- `trademark_data.json` - 商标分类数据
- `scrape_data.py` - 数据抓取脚本

## 数据更新

如需更新数据，运行：

```bash
python3 scrape_data.py
```

## 技术栈

- 纯HTML/CSS/JavaScript，无框架依赖
- 使用现代CSS Flexbox布局
- 响应式设计，支持不同屏幕尺寸

## 注意事项

- 导出的文件格式为CSV（兼容Excel）
- 复制功能使用浏览器剪贴板API
- 数据来源于公开API，仅供学习参考使用
